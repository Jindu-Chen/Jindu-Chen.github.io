记录链接脚本的学习内容

概述

链接脚本的主要作用：描述如何将输入文件(.o目标文件)中的段 映射到 输出文件(.bin/.hex/…)中的段，并控制输出文件的内存布局(地址分配)。

注意：输入文件的段和输出文件的段是独立的，汇编源文件和C源文件中声明的段是输入段，在链接文件中.text: { xxx}的text是输出的段

符号

  • ENTRY
  • MEMORY
  • SECTIONS
  • KEEP
  • LOADADDR
  • PROVIDE
  • PROVIDE_HIDDEN
