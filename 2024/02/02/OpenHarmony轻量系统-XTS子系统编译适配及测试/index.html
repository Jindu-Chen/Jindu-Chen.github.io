д»Ӣз»ҚеҹәдәҺеӣҪж°‘жҠҖжңҜN32G452иҠҜзүҮ(Cortex-M4еҶ…ж ё)зҡ„OpenHarmony-V4.0иҪ»йҮҸзі»з»ҹз§»жӨҚе·ҘзЁӢпјҢиҝӣиЎҢXTSеӯҗзі»з»ҹзј–иҜ‘йҖӮй…ҚеҸҠеә”з”Ёе…је®№жҖ§жөӢиҜ•еҲҶжһҗзҡ„ејҖеҸ‘иҝҮзЁӢпјҢд»ҘдҫӣеҸӮиҖғ

жң¬ж–ҮеҶ…е®№дёәжң¬дәәеңЁз§»жӨҚеӣҪж°‘жҠҖжңҜN32G452йҖӮй…ҚOpenHarmonyиҪ»йҮҸзі»з»ҹзҡ„иҝҮзЁӢдёӯпјҢд»Ҙж·»жІ№ж–№ејҸд»“дҝғеҶҷе°ұпјҢе…¶дёӯжҲ–жңүдёҚе°‘зӣёдә’зҹӣзӣҫжҲ–жңүеӨұеҒҸйўҮд№ӢеӨ„пјҢдҪҶеҜ№дәҺеҲқе…ҘжӯӨйҒ“д№ӢејҖеҸ‘иҖ…еә”иғҪжңүдёҖе®ҡзҡ„еҗҜиҝӘдҪңз”ЁгҖӮ

иҪ»йҮҸзі»з»ҹеә”з”Ёе…је®№жҖ§жөӢиҜ•йҖӮй…Қ

  • еүҚжҸҗпјҡеҹәдәҺOpenHarmony-V4.0-Releaseе…ЁйҮҸд»Јз Ғд»“
  • еңЁvendorзӣ®еҪ•дёӢдә§е“Ғзҡ„config.jsonж–Үд»¶дёӯпјҢж·»еҠ XTSеӯҗзі»з»ҹз»„д»¶е®ҡд№ү

еҰӮдҪ•ж·»еҠ XTSеӯҗзі»з»ҹ

жү§иЎҢзј–иҜ‘XTSз»„д»¶зҡ„жҢҮд»ӨдёҚе”ҜдёҖпјҢеҸ–еҶідәҺgnжһ„е»әж–Үд»¶еҶ…зҡ„з§Ғжңүе·ҘзЁӢи®ҫзҪ®

  • йҖҡеёёдёәпјҡhb build -f -b debug --gn-args build_xts=trueпјҢbuild_xtsжҳҜдёҖдёӘеҸҳйҮҸпјҢеҰӮжһңдҪҝиғҪдәҶиҜҘеҸҳйҮҸпјҢеҲҷдјҡеўһеҠ зӣёе…ізҡ„зј–иҜ‘йғЁеҲҶгҖӮеҸҰеӨ–еңЁз”ЁжҲ·з§Ғжңүзҡ„BUILD.gnж–Үд»¶йҮҢпјҢд№ҹжңүиҜҘеҸҳйҮҸзҡ„зӣёе…ідҫқиө–йғЁеҲҶгҖӮ--gn-argsз”ЁдәҺдј йҖ’жһ„е»әеҸӮж•°иҮіGNжһ„е»әзі»з»ҹ
  • hb build -b debugпјҢжү§иЎҢжӯӨжқЎжҢҮд»Өз”ҹжҲҗи°ғиҜ•зүҲжң¬й•ңеғҸпјҢдёҺ XTS зј–иҜ‘ж— е…і

е°ұжң¬дәәзӣ®еүҚзҗҶи§ЈиҖҢиЁҖпјҢиҰҒж·»еҠ XTSеӯҗзі»з»ҹпјҢеүҚжңҹйңҖиҰҒе®һзҺ°зҡ„еҮ зӮ№е·ҘдҪңеҰӮдёӢпјҡ

  • йҰ–е…Ҳе®ҢжҲҗеҹәдәҺз§ҒжңүејҖеҸ‘жқҝзҡ„ OpenHarmony е·ҘзЁӢз§»жӨҚйҖӮй…Қе·ҘдҪңпјҢдҪҝеҫ—еҹәдәҺеҶ…ж ёеӯҗзі»з»ҹзҡ„жңҖз®Җе·ҘзЁӢеҸҜд»ҘиҝҗиЎҢ
  • иҖҢеҗҺе»әи®®еңЁtarget_config.hж·»еҠ еҗҜеҠЁshellеҠҹиғҪй…ҚзҪ®пјҢеҗҢж—¶е®ҢжҲҗдёІеҸЈеӨ–и®ҫи°ғиҜ•з»Ҳз«Ҝзҡ„з§»жӨҚйҖӮй…ҚпјҢhb buildпјҢе№¶зғ§еҪ•йЎ»жөӢиҜ•йӘҢиҜҒokгҖӮеҪ“з„¶жңҹй—ҙеҶ…еҝ…еӯҳеңЁеҗ„з§Қеҗ„ж ·зҡ„зј–иҜ‘жҠҘй”ҷпјҢеҢ…жӢ¬дҫқиө–е…ізі»дёҚе…ЁгҖҒд»Јз Ғе·ҘзЁӢзҡ„дёҚйҖӮй…Қзӯү
  • ж·»еҠ еҗ„йЎ№еӯҗзі»з»ҹе№¶дҪҝе…¶жһ„е»әжҲҗеҠҹпјҢеҰӮеҗҜеҠЁжҒўеӨҚеӯҗзі»з»ҹгҖҒDFXеӯҗзі»з»ҹпјҢзӯүзӯүпјҢ
  • жңҖеҗҺж·»еҠ XTSеӯҗзі»з»ҹй…ҚзҪ®еҸҠе…¶дҫқиө–пјҢеҪ“з„¶жөӢиҜ•еҘ—д»¶зҡ„йқҷжҖҒеә“ж–Үд»¶зӣёеҪ“еӨ§пјҢеҸҜд»ҘеҲҶжү№жөӢиҜ•е№¶дҝқеӯҳжөӢиҜ•з»“жһңпјҢжҜҸж¬Ўй“ҫжҺҘдёҖйғЁеҲҶеә“иҝӣиЎҢжөӢиҜ•
    • дёҖиҲ¬еңЁ//device/board/xxx/xxx/BUILD.gnж–Үд»¶йҮҢзҡ„ if(build_xts=true)йҖүйЎ№еҚЎеҶ…ж·»еҠ зӣёе…ізҡ„й“ҫжҺҘйҖүйЎ№пјҢе…¶дёӯhctestгҖҒsysparamгҖҒbootstrapзӯүйқҷжҖҒеә“жҳҜеҝ…йЎ»зҡ„пјҢеңЁж·»еҠ XTSеӯҗзі»з»ҹд№ӢеүҚпјҢеә”е®ҢжҲҗзӣёе…ійғЁеҲҶйғЁд»¶зҡ„зј–еҶҷзј–иҜ‘жһ„е»әдёҺжөӢиҜ•пјҢжӯӨйғЁеҲҶжҳҜжөӢиҜ•жЎҶжһ¶еҹәзҹі

ж·»еҠ зј–иҜ‘XTSеӯҗзі»йңҖиҰҒжү§иЎҢзҡ„жӯҘйӘӨпјҡ

  • еңЁе·ҘзЁӢй…ҚзҪ®target_config.hе®Ҹе®ҡд№үдёӯдҪҝиғҪTestжөӢиҜ•йЎ№
    • дҪҝеҫ—зі»з»ҹеҗҜеҠЁеҲқе§ӢеҢ–ж—¶дјҡиҝӣиЎҢзӣёе…іеҮҪж•°и°ғз”Ё
  • еңЁ//kernel/liteos_mдёӯmake menuconfigдёӯпјҢдҪҝиғҪжөӢиҜ•зӣёе…ій…ҚзҪ® вҖ“ е®һйҷ…дёҠдёәй…ҚзҪ®//vendor/xxx/kernel_configs/debug.configйҖүйЎ№
  • еңЁ//vendor/xxx/config.jsonж·»еҠ еҗҲйҖӮзҡ„еӯҗзі»з»ҹй…ҚзҪ®
    • иЎҘе…Ёж–Үд»¶жң«е°ҫзҡ„third_patryе’Ңproduct_adapt_dir

    • й“ҫжҺҘйҖүйЎ№дёӯеўһж·»еҗҲйҖӮзҡ„йқҷжҖҒеә“й“ҫжҺҘпјҢд»Ҙе®һзҺ°й“ҫжҺҘжөӢиҜ•з”ЁдҫӢзӣёе…іеә“пјҢе…¶дёӯжөӢиҜ•йқҷжҖҒеә“ж–Үд»¶з”ҹжҲҗдәҺ//out/${company}//${product}//libs
    • иҝҳйЎ»ж·»еҠ жәҗж–Үд»¶и·Ҝеҫ„пјҹ

  • йҖҡиҝҮhbе·Ҙе…·иҜ»еҸ–зі»з»ҹй…ҚзҪ®ж–Үд»¶(еҚіvendorзӣ®еҪ•дёӢдә§е“Ғзҡ„config.json)пјҢиҺ·еҸ–еҸӮдёҺзј–иҜ‘зҡ„еӯҗзі»з»ҹд»ҘеҸҠйғЁд»¶дҝЎжҒҜ
