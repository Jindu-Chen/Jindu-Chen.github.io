## 杰理SDK架构解析


### SDK目录概览

以SDK包为根，主要有以下子目录
```
//SDK
│
│
├── apps
│      │
│      ├── common         # 通用中间层文件，如音频、蓝牙、全局配置、设备管理、文件系统等
│      │
│      └── soundbox       # 应用层的业务逻辑代码
│
├── cpu/br25             # 内核、芯片外设层的相关代码
│      │
│      ├── audio_common/
│      │
│      ├── tools/        # 工具集，包含有配置工具、烧录脚本等
│      │
│      └── ...           # 
│
│
└── include_lib          # 封装好，以供调用或者链接的库，包含OS、FS、外设配置等

```
