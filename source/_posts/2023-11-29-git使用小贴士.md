# 子模块

- 删除项目内已有的子模块配置，将其转化为本项目内容
```    
cd ./path/to/submodule
rm -rf .git

# at project root directory
git rm --cached ./path/to/submodule
git add --all
git commit -m "xxx"
git push
```
